Task 1
-------

Для работы требуется библиотека GD.
Работающий пример можно посмотреть здесь http://yell.stoiss.net/task-1/

Основные моменты:

1. Все фигуры наследуются от абстрактного класса Shape;
2. Board - класс "доски" для рисования, на которой затем рисуются фигуры;
3. Builder - реализация порождающего шаблона проектирования. Класс отвечает 
за создание экзмемляра board и "рисования" фигур потомками Shape исходя из настроек.

app.php - само приложение. В get параметре "shapes" принимает serialize массив с данным 
фигур и выводит получившееся изображение в формате png.

index.php - фронтенд для тестирования.

AppExtension.php - пустой класс, наследник Extension. 
"Бросаю" в него ошибки, если такие возникают.


Фронтенд часть сделал максимально простой. Насколько я понял, она не принципиальна для
задания.
